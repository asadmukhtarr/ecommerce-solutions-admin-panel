# 🚀 Enterprise E-Commerce Admin Dashboard

> A modern **Laravel + Livewire** administration panel for managing an online store from a single dashboard.

## 🌟 Overview

Enterprise E-Commerce Admin Dashboard is a feature-rich administration system built with **Laravel, Livewire, Bootstrap 5, JavaScript and MySQL**. It centralizes products, orders, customers, payments, shipping, reports, website content, and system settings into one responsive interface.

## ✨ Features

- Dashboard Analytics
- Orders Management
- Products, Categories & Brands
- Inventory Management
- Customers Management
- Coupons & Flash Sales
- Payments & Transactions
- Shipping Zones & Methods
- Reports & Analytics
- User Roles & Permissions
- Website CMS (Slider, Pages, Banners, Blog, SEO)
- General Settings & Payment Gateways
- Livewire powered navigation
- Responsive Bootstrap 5 UI

## 🛠 Tech Stack

- Laravel
- Livewire
- PHP
- MySQL
- Bootstrap 5
- JavaScript
- HTML5
- CSS3
- Font Awesome

## ⚙️ Installation

```bash
git clone https://github.com/asadmukhtarr/ecommerce-solutions-admin-panel.git
cd ecommerce-solutions-admin-panel
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm run build
php artisan serve
```
